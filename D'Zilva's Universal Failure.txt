D’Zilva’s Universal Failure Framework
Extending Ashby’s Law of Requisite Variety into Time, Integrity, and Control Collapse

Abstract
Ashby’s Law of Requisite Variety establishes that a system can only remain in control if its regulatory capacity matches or exceeds the complexity of disturbances it faces:
R(S) >= R(D)
While foundational, this formulation assumes:
A stable regulator
Reliable feedback
No degradation over time
These assumptions do not hold in real-world systems.

D’Zilva’s Universal Failure Framework extends this principle by introducing:
Time dependence
Integrity constraints
Independence of oversight
Signal degradation
Control is therefore not a static condition, but a continuously evaluated inequality subject to collapse.

1. From Static Control to Dynamic Failure
Ashby (1956):
R(S) >= R(D)
Where:
R(S) = Regulatory variety (system capacity)
R(D) = Disturbance variety (environment complexity)

This assumes:
A stable regulator
Clean feedback loops
No structural decay
In reality, systems degrade.

2. The D’Zilva Extension (2026)
The framework introduces the following variables:
kappa (k) = Independence Coefficient
Degree to which the correction mechanism exists outside the distortion field
C = Correction Capacity
Internal ability to respond to valid signals
Theta(C - C_crit) = Allee Threshold Function
Nonlinear collapse condition
DeltaI (DI) = Irreversibility Index
Accumulated structural lock-in over time
S(t) = Signal Suppression
Distortion or loss of corrective feedback
Extended Control Condition
R(S, t) * k * Theta(C - C_crit) >= R(D, t) + DeltaI(t) + S(t)

3. Effective Control Condition
From implementation, effective control becomes:
R_control(t) >= L_load(t)
Where:
R_control(t) = R(S, t) * k * Theta(C - C_crit)
L_load(t) = R(D, t) + DeltaI(t) + S(t)
Key Insight
A system is only controllable if:
Correction capacity exists
Correction capacity is independent
Signals are not suppressed
High capacity without independence results in illusory control.

4. Observer Collapse
Observer Collapse occurs when:
k -> 0
This represents loss of independence in oversight.
Effects:
Signals become untrustworthy
Correction becomes self-referential
The system cannot detect its own failure
This is a gradient condition:
Partial collapse -> degraded correction
Full collapse -> total loss of observability

5. The Zombie Zone
A system enters the Zombie Zone when:
DI_effective > 1
AND
C > C_min
Where:
DI_effective = distortion adjusted for correction capacity
Characteristics:
System continues operating
Alignment is lost
Distortion is masked as performance
Intervention cost rises rapidly

6. Failure as a Trajectory
Failure is not an event — it is a trajectory.
Defined by:
Distortion Velocity (dDI/dt)
Distortion Acceleration (d²DI/dt²)
This allows:
Early detection
Predictive modelling
Intervention before collapse

7. System State Representation
State(t) = [DeltaI(t), S(t), k(t), C(t)]
Effective correction:
C_eff = C * k

8. The Master Inequality
At every moment:
R_control(t) >= L_load(t)
Control exists only while this condition holds.

9. Implications
This framework demonstrates:
Scaling capacity alone does not ensure control
Integrity must remain stable under variance
Independence is a first-order requirement
Failure begins before visible collapse

10. Toward Operationalisation
The framework defines the minimum variables required to:
Measure distortion
Detect failure modes
Predict collapse
Enable intervention
This enables transition from:
Static compliance
to
Real-time structural integrity monitoring
Closing Statement
Ashby defined the condition for control.
D’Zilva defines the condition for loss of control under real-world constraints.

Control is not lost when capacity falls behind complexity.
Control is lost when:
Independence degrades
Signal integrity collapses
Irreversibility accumulates
Correction capacity becomes ineffective
